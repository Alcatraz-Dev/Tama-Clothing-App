# Bey3a Landing Page

Modern landing page for the Bey3a fashion app.

## Tech Stack

- React 18
- TypeScript
- Vite
- Lucide React (icons)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Customization

- Edit `src/App.tsx` to modify content
- Edit `src/index.css` to change styles
- Colors are defined as CSS variables in `src/index.css`

## Branding

- Primary Color: #000000
- Accent Color: #FF6B6B
- Font: Inter (system fonts fallback)
